# /ventes — Contrôle d'accès

## Agents autorisés à écrire dans /ventes/opportunities/

| Agent | Raison |
|---|---|
| IT-FrontLine | Triage — détection besoins client en N1 |
| IT-Assistant-N2 | Support — identification besoins complexes |
| IT-Assistant-N3 | Diagnostic avancé — besoins infra/projet |
| IT-SysAdmin | Interventions terrain — upgrade/migration identifiés |
| IT-MaintenanceMaster | Maintenance — lacunes infra détectées |
| IT-OnOffBoarder | Onboarding — analyse lacunes → projets |
| IT-NetworkMaster | Réseau — projets firewall, VLAN, VPN |
| IT-SecurityMaster | Sécurité — projets remédiation, EDR, SOC |
| IT-BackupDRMaster | Backup — projets DR, Veeam, Datto |
| IT-CloudMaster | Cloud — projets Azure, M365, migration |

## Agent lecteur (read + generate)
| Agent | Rôle |
|---|---|
| IT-ProjetSOW | Lit les opportunités, génère estimations et SOW |

## Validation finale
Tout prix soumis au client requiert validation EA avant envoi.
