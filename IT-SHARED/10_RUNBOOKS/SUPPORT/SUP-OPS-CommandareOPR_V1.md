## 1. CLOTURE DE BILLET CW

### CW Note Interne (interne MSP)
Premiere ligne OBLIGATOIRE :
```
Prise de connaissance de la demande et consultation de la documentation du client.
```
Puis : chronologie des actions, resultats, observations techniques.

### CW Discussion (visible client sur facture)
Deux premieres lignes OBLIGATOIRES :
```
Preparation et decouverte.
Prendre connaissance de la demande et connexion a la documentation de l'entreprise.
```
Puis : TRAVAUX EFFECTUES (liste puces, min 4 items, langage client).

## 3. DEFINITION OF DONE (DOD)

Un billet est DONE quand :
- [ ] Probleme resolu ou escalade formellement
- [ ] CW Note Interne completee
- [ ] CW Discussion completee (facturable)
- [ ] Client notifie si impact P1/P2
- [ ] Hudu mis a jour si infra modifiee
- [ ] KB cree si nouveau type d'incident
- [ ] Temps correctement impute dans CW
