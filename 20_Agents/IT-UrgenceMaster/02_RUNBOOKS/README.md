# Runbooks — IT-UrgenceMaster

Ce dossier contient les procédures pas à pas utilisées par l’agent pour piloter une urgence de façon reproductible.

## Runbook principal inclus
- `RB-001_procedure-principale.md` : flux principal d’une urgence P1/P2, de la qualification à la clôture.

## Structure attendue d’un runbook
- objectif ;
- déclencheur ;
- entrées minimales ;
- séquence opératoire ;
- critères de décision ;
- sorties attendues ;
- erreurs fréquentes / fallback.
