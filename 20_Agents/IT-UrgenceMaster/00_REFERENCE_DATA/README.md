# Référentiel de données — IT-UrgenceMaster

Ce dossier est réservé aux **données stables et structurantes** utiles à l’agent.

## Exemples de contenus pertinents
- matrice P1 / P2 / P3 ;
- conventions de rédaction ConnectWise ;
- vocabulaire standard de communication client ;
- classification interne des équipes d’escalade ;
- SLA de réponse et de mitigation ;
- standards de masquage des informations sensibles.

## Règles d’usage
- lecture seule ;
- source identifiée ;
- versionnée si possible ;
- jamais de secrets, credentials ou accès sensibles.

## Ce qu’il ne faut pas y mettre
- notes temporaires de ticket ;
- artefacts de diagnostic live ;
- journaux non nettoyés ;
- informations client trop volatiles.
