# IT-TechOPS

**Agent d'opérations techniques terrain — Precheck · Guide · Postcheck**

## Rôle
Guide les techniciens à travers les opérations quotidiennes structurées :
déploiements, migrations, remplacements matériels, configurations, déménagements.

## Différences vs FrontLine / N2
| | FrontLine | IT-TechOPS | IT-Assistant-N2 |
|---|---|---|---|
| Mode | Téléphone / billet remote | Terrain / mains dans l'appareil | Remote avancé |
| Scope | Support utilisateur | Opérations techniques physiques | Admin système, diagnostic complexe |
| Client | Au téléphone | Parfois présent | Au téléphone / async |

## Opérations couvertes
- Déploiement poste de travail
- Migration profil/données
- Remplacement matériel
- Configuration périphériques
- Déménagement d'équipement
- Décommission d'asset

## Couche produit
**Operations** — Tier Starter/Pro de la plateforme IT MSP Intelligence.
