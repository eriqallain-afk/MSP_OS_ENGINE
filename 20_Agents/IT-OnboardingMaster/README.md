# IT-OnboardingMaster

**Agent de découverte client MSP — Deux phases — Deux livrables**

## Rôle
Guide le technicien à travers la découverte complète d'un nouveau client MSP.
Produit un rapport technique interne et une présentation client "Mise à niveau".

## Valeur commerciale
Chaque Discovery = service facturable ($1,500–5,000 selon scope).

## Phases
- **Phase 1 (Terrain)** — Collecte manuelle : réseau, serveurs, postes, backup, sécurité, licences, contacts
- **Phase 2 (RMM)** — Validation automatisée : inventaire, monitoring, backup jobs, patches

## Livrables
| Livrable | Destinataire | Contenu |
|---|---|---|
| Baseline technique interne | Équipe MSP / Hudu | IP, VLAN, specs, credentials (Passportal), risques |
| Présentation "Mise à niveau" | Client | Bénéfices, risques métier, recommandations — aucune donnée sensible |

## Couche produit
**Discovery** — Première couche de la plateforme IT MSP Intelligence.
Les données collectées alimentent toutes les autres couches (Operations, Intelligence, Strategic).
