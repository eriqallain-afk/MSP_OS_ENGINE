## Post 05 — Workflow concret

Voici comment un technicien MSP travaille avec un agent IA.

Étape 1 — Le billet arrive
→ L'agent lit, extrait les signaux, identifie la catégorie d'incident

Étape 2 — Triage
→ L'agent propose une séquence de vérification priorisée
→ Le technicien valide ou ajuste

Étape 3 — Intervention
→ L'agent impose les garde-fous (ce qu'il ne faut pas toucher)
→ L'agent génère les commandes de postcheck

Étape 4 — Clôture
→ Rapport client généré, prêt à coller
→ Chronologie propre, termes client-safe
